# عزم - برنامج الإنجاز والتشغيل

برنامج متابعة مهام وتشغيل داخلي لشركة عزم اللوجستية.

## المميزات

- لوحة تحكم شاملة مع مؤشر جاهزية الانطلاق
- خارطة مراحل التأسيس والتشغيل
- لوحة Kanban للمهام مع سحب وإفلات
- نظام الحضور والانصراف
- تتبع المعوقات
- التقارير اليومية والأسبوعية
- نظام تنبيهات داخلي
- المساعد التشغيلي

## المستخدمون

- **الأستاذ سلطان**: المدير العام (صلاحيات كاملة)
- **عبدالرحمن**: مدير العمليات (صلاحيات محدودة)

## تسجيل الدخول

- المدير العام: `sultan@azm.sa` / `azm2026`
- مدير العمليات: `abdulrahman@azm.sa` / `azm2026`

## التثبيت

```bash
npm install
npm run build
```

## النشر

المشروع جاهز للنشر على Hostinger. اتبع الخطوات التالية:

1. ارفع الملفات من مجلد `dist` إلى Hostinger
2. تأكد من إعداد SPA routing correctly

## التقنيات

- React 18
- Vite
- Tailwind CSS
- React Router 6
- @dnd-kit for drag and drop
- PWA support
